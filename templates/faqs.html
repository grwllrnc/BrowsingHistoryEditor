FAQs

Wie funktioniert das Programm?

Das Programm ist ein Browser-basiertes Tool, das wie eine Webseite funktioniert. Wenn Sie das Programm ausführen, wird ein lokaler Webserver auf Ihrem Computer gestartet (namens ‘localhost’), der auf dem Port 5000 läuft. (Folgende Adresse wird im Browser angezeigt: http://localhost:5000). Dabei öffnet sich ein Browser-Fenster, in dem das Programm läuft.

Werden bei der Nutzung des Programms Daten über das Internet übermittelt?

Nein, das Programm wird lokal auf Ihrem Computer ausgeführt und es werden keine Daten über das Internet übertragen.

Wie werden die Einträge anonymisiert?

Die URL wird mit einem zufälligen Unique-Identifier ersetzt, der keinerlei Rückschlüsse auf die URL zu lässt und nicht entschlüsselt werden kann. Suchbegriffe werden ebenfalls mit einem zufälligen Unique-Identifier ersetzt. Beispiel: 

anonymisiert-968c9d6056f7a8efcd045b61ea9848e392dc4a018378259c9a5e758fd71a6e5e-ec749af98d4d40389b894816dbb2568e

Der Titel der Webseite, der Host und die Redirect URL werden durch ‘***’ ersetzt.

Mit der Funktion 'URLs kürzen' haben Sie die Möglichkeit, den URL-Teil (rot markiert) nach der Domain abzuschneiden: 'http://www.meteoschweiz.admin.ch/home/aktuell/meteoschweiz-blog/meteoschweiz-blog.html?topic=/content/meteoswiss/tags/topics/wetter'

Warum kann ich die Einträge des Browserverlaufs nur anonymisieren und nicht löschen?

Das Programm dient dazu, den Umfang Ihrer Internetnutzung zu messen. Dabei soll die Nutzung von bestimmten Webseiten im Verhältnis zu Ihrer gesamten Internetnutzung möglichst genau bestimmbar sein. Durch das Löschen von Einträgen des Browserverlaufs würde das Verhältnis unter Umständen verfälscht werden.

Werden alle Einträge aus dem Browserverlauf geladen bzw. exportiert?

Nein, der Daten-Umfang ist zeitlich beschränkt. Die Software lädt nur die Einträge der letzten 60 Tage.

Kann das Programm auf Daten aus dem privaten oder Inkognito-Modus zugreifen?

Nein, diese Daten werde nicht im Browserverlauf gespeichert.

Welche Daten werden exportiert?

Welche Daten exportiert werden, hängt von dem Browser ab, den Sie benutzen. Nicht alle Browser speichern die gleichen Daten. Je nach Browser werden folgende Daten exportiert:

  • URL (z.B. https://www.google.ch/#q=nachrichten+schweiz+heute)
  • URL-ID (eine Zahl, mit der die URL eindeutig identifiziert werden kann)
  • Visits-ID (ID des Besuchs)
  • Titel der URL (z.B. ‘nachrichten schweiz heute - Google-Suche’)
  • URL-Weiterleitungen
  • Referrer (die Visit-ID der URL, von der Sie auf die aktuelle URL gekommen sind)
  • Datum und Uhrzeit des Seitenaufrufs (als Zeitstempel)
  • Datum und Uhrzeit des letzten Besuchs (als Zeitstempel)
  • Domain (z.B. ‘google.ch’)
  • Häufigkeit der Besuche
  • Informationen, ob die URL eingetippt wurde oder nicht
  • Wie die URL besucht wurde (z.B. via Link)
  • Name des Browsers
  • Name des Betriebssystems

Wenn Sie den Browserverlauf exportieren, wird eine CSV-Datei generiert, die Sie z.B. in Excel öffnen können.

Bitte nehmen Sie keine Veränderung an der CSV-Datei vor. Falls Sie nach dem Export noch Änderungen machen möchten, gehen Sie bitte folgendermassen vor:

  1. Öffnen Sie das Programm erneut.
  2. Nehmen Sie die gewünschten Änderungen vor.
  3. Exportieren Sie die Daten noch einmal.

Werden Zugangsdaten wie Benutzernamen oder Passwörter exportiert?

Nein, es werden keine Zugangsdaten verarbeitet oder exportiert.

  • Kurze Problembeschreibung

  • Version des Betriebssystems

Kann ich den Quellcode des Programms einsehen?

Ja. Der Quellcode ist auf Github zu finden.

Wird der Browserverlauf meines Browsers durch die Software verändert oder überschrieben?

Nein, der Browserverlauf bleibt unverändert. Das Programm erstellt eine lokale, temporäre Kopie Ihres Browserverlaufs. Wenn Sie Einträge des Browserverlauf mit dem Programm bearbeiten (anonymisieren und exportieren), dann werden diese Änderungen an der Kopie durchgeführt. Wenn Sie das Programm von Ihrem Computer entfernen (löschen), wird auch die Kopie gelöscht.

Verändert die Software Dateien, Ordner oder Programme auf meinem Computer?

Nein, es werden weder eigene noch System-Dateien oder -Ordner verändert. Alle Dateien und Ordner, die das Programm verwendet, sind in dem Ordner des Programms gespeichert. Dort werden auch die lokalen, temporären Dateien erstellt (Kopien des Browserverlaufs als Datenbank und als CSV-Datei). Wenn Sie das Programm wieder entfernen (den Ordner des Programms löschen), werden auch diese Dateien gelöscht.

Was passiert mit meinen Daten?

Die Daten des Browserverlaufs werden vertraulich behandelt und ausschliesslich im Rahmen des Forschungsprojekts verwendet. Sie werden nicht an Dritte weitergegeben, und Dritte haben keine Einsicht in die Daten. Die Daten werden nur in anonymisierter Form für Publikationen und Präsentationen im Rahmen des Forschungsprojekts verwendet, so dass keinerlei Rückschlüsse auf Ihre Person möglich sind.

Warum muss ich das Programm als Administrator ausführen, wenn ich den Browserverlauf des MS Internet Explorers / Edge bearbeiten möchte?

Der Internet Explorer (ab Version 10) speichert den Browserverlauf in der Datei WebCacheV01.dat. Diese Datei wird von Windows geöffnet und für andere Programme gesperrt, sobald Sie ihren Computer einschalten bzw. sich anmelden. Damit der Browserverlauf gelesen werden kann, erstellt die Software eine lokale Kopie (eine so genannte Volume Shadow Copy) der Datei erstellt. Damit der dafür zuständige Windows-Dienst Volume Shadow Copy Service (vss) genutzt werden kann, muss das Programm mit Administrator-Rechten ausgeführt werden.

Wie öffne ich das Programm mit Administrator-Rechten?
Im Kontextmenü (klicken Sie mit der rechten Maustaste auf die Programm-Datei) wählen Sie die Option „Als Administrator ausführen“.
